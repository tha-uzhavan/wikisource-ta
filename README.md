எனக்கு நூல்கள் வாசிப்பது, மறுகட்டமைப்பு செய்வது பிடிக்கும். அன்றாடம் உணவு உண்ணுவது போல, என் மனதிற்கு நூல்கள் மிகவும் அடிப்படையாகும். இங்கு நான் விக்கிமூலம் என்ற கட்டற்ற தளத்தில் செய்துவரும் முயற்சிகளை இடுகிறேன். நான் முறைப்படி கல்லூரியில் பைத்தான் படித்தவன் அல்ல. தேவையின் பொருட்டு இணையதளங்களில் இருந்தும், பிறரிடம் கேட்டும் இங்குள்ளவை தேவையின் பொருட்டு உருவாக்கியுள்ளேன். எனது தேவைகள் முடிந்தன என்றாலும், மறுவாசிப்புக்காக இவற்றினை இங்கு சேமிக்கிறேன். நீங்கள் இதனை விட எளிமையான வழிமுறைகளைக் கட்டமைத்தால் எனக்கு அறியத் தாருங்கள். ஏறத்தாழ 60 வயதை நெருங்கும் எனக்கு நிறைய நேரத்தினை மீதமாக்கினால் எனக்கு மட்டற்ற மகிழ்ச்சியே. விக்கிமூலம்:பைத்தான் நிரல்கள் = https://ta.wikisource.org/s/9z0e  இந்த பக்கத்தில் முன்பு இட்ட நிரல்களையே, இங்கும் இடுகிறேன். இருப்பினும் அவ்வப்போது புதிய நிரல்கோப்புகளையும் இங்கு உருவாக்குவேன்.
